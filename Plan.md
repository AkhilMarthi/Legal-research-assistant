# Project

Build a **Legal Research Assistant** using Retrieval-Augmented Generation (RAG).

The application should allow users to:

* Upload legal documents (PDF, Markdown, TXT)
* Index those documents
* Ask legal questions in natural language
* Retrieve relevant sections
* Generate answers using an LLM
* Cite the exact document, page, and section used
* Stream responses like ChatGPT

Everything must run locally.
